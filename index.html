About & Learning Goals

I'm a software engineering student building Android apps with Kotlin and Jetpack Compose. The goal of this semester was to create an app from static UI screens to a fully connected app, an app that stores data locally, syncs to the cloud, pulls live data from the internet, and uses device hardware properly. This portfolio is collection of all my lab work and two project reports built around SDG 4: Quality Education.

Reflection & Learning Journey

I started this class unable to lay out a screen and unsure how apps cam actually hold and move data. But after building Kouleej! I started to unstand the gist of Mobile Application Development. Project 1 taught me to think in terms of navigation and a single shared ViewModel and Project 2 forced me to connect that state to four very different data sources. A local Room database, Firebase Firestore in the cloud, a live REST API through Retrofit, and the camera via ML Kit. The hardest part was just inconsistencies in UI and debugging random parts of my code because it kept breaking. But the payoff for SDG 4 is direct now a learner with a cheap phone can now build a quiz, share it, and let a whole class join by scanning a QR code with no logins or links required.
